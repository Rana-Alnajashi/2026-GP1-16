<h1>Nafas – Smart Asthma Monitoring System</h1>

<h2>Overview</h2>

<p>Nafas is an IoT-based smart asthma monitoring system designed to support continuous respiratory health awareness in children.

The system integrates wearable physiological and environmental sensors with cloud-based processing and a mobile application to provide real-time monitoring and contextual health recommendations for caregivers.

Nafas collects health indicators through a wristband device, stores and processes the data in the cloud, and delivers alerts and weather-based recommendations through a secure iOS application.</P>

<h2>Project Goal</h2>
<p>To develop an intelligent respiratory monitoring system that:</p>
<ul>
  <li>Enables continuous health monitoring using wearable sensors</li>
  <li>Provides environmental-based recommendations using weather data</li>
  <li>Supports early awareness of abnormal physiological changes</li>
  <li>Integrates cloud-based data processing and storage</li>
  <li>Assists parents in proactive asthma management</li>
  <li>Aligns with Saudi Vision 2030 digital health transformation goals</li>
</ul>

<h2>System Architecture</h2>
<p>Nafas follows a four-layer IoT architecture:</p>
<ul>
  <li><Strong>Perception Layer – </Strong>Wearable sensors collect physiological and environmental data</li>
  <li><Strong>Network Layer –</Strong> BLE and Wi-Fi communication</li>
  <li><Strong>Data Processing Layer –</Strong> Firebase storage & Google Cloud model execution</li>
  <li><Strong>Application Layer –</Strong> iOS app interface & weather integration</li>
</ul>
